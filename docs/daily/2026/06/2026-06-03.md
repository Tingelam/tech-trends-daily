### 1. [chopratejas/headroom](https://github.com/chopratejas/headroom) `Python` ⭐6,432 +1,265

**核心功能：** 一个专门压缩工具输出、日志、文件与 RAG chunk 的上下文治理层，可作为库、代理或 MCP server 使用。它瞄准的不是“再做一个模型”，而是把进入 LLM 的原始上下文先做高压缩率整理，从而降低成本并维持回答质量。

**技术创新：**

- **上下文前置压缩**：项目描述直接给出 `60-95%` token 降幅，定位非常明确——先压缩，再推理。
- **多接入形态**：同时提供 library / proxy / MCP server，说明它希望嵌入不同 Agent 栈，而不是绑定单一框架。
- **对 Agent 成本结构有直接影响**：当多工具链输出越来越长时，压缩层会比单纯拉长 context window 更有工程价值。
